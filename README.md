
# 项目概述

本项目为旅行地图规划助手，旨在为旅行者提供优化的行程规划方案。项目运用了Python和HTML技术开发，结合算法优化，以解决旅行规划中的实际问题。

项目代码仓库：https://github.com/PaluAix/Simple-graph-calculator

## 系统要求

- Python版本：3.9或更高

## 本地部署指南

1. 确保安装必要的依赖：
   ```
   pip install Flask
   ```
2. 执行 `app.py` 启动应用。

3. 使用浏览器访问 `http://127.0.0.1:8888/` 进入应用界面。

4. 按照指示上传数据进行处理。
