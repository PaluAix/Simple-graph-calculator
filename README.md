# 作品概述

这是旅行地图规划助手，可以为外出旅行的人提供最佳攻略

本作品采用 Python 和 HTML 编写

在技术上，本作品与信息竞赛紧密结合，利用算法解决实际问题

仓库地址：https://github.com/PaluAix/Simple-graph-calculator

## 运行环境

- Python 版本需在 3.9 以上

## 本地部署

1. 首先，确保已安装以下依赖：
   ```
   pip install Flask
   ```

2. 运行 app.py。

3. 在本地打开浏览器，并访问 `http://127.0.0.1:8888/`，即可进入服务页面。

4. 上传需要处理的数据
